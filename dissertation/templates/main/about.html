What is this website about?

This website aims to teach you the basics of Java programming. This involves learning 3 different types of conditional statements. The program uses AI to gauge an understanding of your strengths and weaknesses and uses this to assign you task to help you learn the material correctly.

How should you complete this course?

First complete the learning style and the pretest. The learning style test will help identify your learning style. Research has shown that people learn better using different methods, this program aims to achieve this and will assign you the correct formatted content and questions to help you learn more efficiently. The pretest is designed to gauge an understanding of your current knowledge state, this will help the systme understand what questions it has to assign you.

What do you need to complete this course?

The program uses a built-in IDE from Repl, any issues with it you can make your own account for free at their website.
